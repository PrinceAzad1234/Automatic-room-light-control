# 💡 Automatic Room Light Control System

## 📌 Project Overview
The **Automatic Room Light Control System** is a smart automation project that automatically turns room lights ON and OFF based on human presence.  
This project helps reduce electricity wastage and improves energy efficiency by eliminating the need for manual switching.

It is suitable for homes, offices, classrooms, and smart building applications.

---

## 🎯 Objectives
- To automate room lighting
- To reduce power consumption
- To develop a low-cost smart lighting solution
- To learn IoT and microcontroller-based automation

---

## 🛠️ Components Used
- NodeMCU (ESP8266) / Arduino
- Ultrasonic Sensor (HC-SR04)
- Relay Module
- Light Bulb
- Power Supply
- Connecting Wires
- Breadboard

*(Edit this list if your components are different)*

---

## ⚙️ How It Works
1. The ultrasonic sensor detects the presence of a person in the room.
2. The microcontroller processes the sensor data.
3. If a person is detected, the relay turns the light ON.
4. If no person is detected, the light turns OFF automatically.
